### Project: Create a Flask web app that recognizes 5 flower types using Deep Learing, deployed with Docker Containers.

### Components:  

Back-End: MySQL  
Classification Model: Modified Keras Pre-trained model (VGG19)  
Front-End: Flask, HTML, CSS  
Deployment: Docker
